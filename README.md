# Terraform-learning
Provisioning infrastructures on AWS using terraform
#steps to follow
create an IAM programmatic user
connect to this user on the aws cli
clone the project to your remote desktop
create a main.tf file in your working directory
edit the code and specify all resources in the main.tf file
run terraform init to initialize the project
run terraform validate to check for errors
run terraform plan to see all resources that you want to create
run terraform apply to create the infrastructure
run terraform destroy to delete or terminate the infrastructure (optional)
